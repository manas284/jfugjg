# **App Name**: EduPrep Pro

## Core Features:

- Dashboard: Displays personal performance stats (Tests, Score, Accuracy, Rank), featured tests carousel, recent activity log, performance chart and quick start options.
- Practice Tests Library: Allows users to filter by exam type (UPSC, SSC, Banking, GATE) and sort by difficulty, duration, and ratings. Displays test cards with detailed information and includes a search functionality.
- Test Interface: Provides an exam mode interface with a question navigator (color-coded status: Green = Answered, Yellow = Marked, Gray = Not visited, Red = Not answered).
- Results Page: Displays overall score, performance percentile, subject-wise breakdown, detailed answer analysis, achievement badges, and options to download & share results.
- Analytics Dashboard: Presents a score trend chart, category-wise performance, test history table, and identification of weak areas.
- AI Performance Tool: Uses an AI tool to analyze user performance and suggests relevant practice tests.

## Style Guidelines:

- Primary color: Strong blue (#2E9AFE) for a sense of professionalism and trust.
- Background color: Light grayish-blue (#EBF5FB) to provide a clean and modern feel.
- Accent color: A vibrant purple (#BE81F7) to highlight important actions and information.
- Body and headline font: 'Inter' for a clean, modern sans-serif feel that ensures readability and professionalism.
- Use a set of consistent and professional icons to represent different exams and features.
- Subtle hover effects and smooth transitions to provide a responsive user experience.
- Responsive layout that works seamlessly on all devices, ensuring accessibility.
